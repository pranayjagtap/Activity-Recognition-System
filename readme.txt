pca.py
The pca.py file takes the generated feature matrix files and performs PCA on them. It generates the eigen vector plot as a .png file and creates 2 csv files for the eigen vectors and the reduced feature matrix.

generate_features.py

The generate_features.py file processes the respective data files and extracts to respective features and arranges them into a matrix for each of the activities. 
It generates 2 csv files cooking_features.csv and eating_features.csv containing the feature matrices for each activity.
